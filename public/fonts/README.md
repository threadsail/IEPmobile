# Fonts

Add **Aloja.woff2** here for the "mobile" text in the header to use the Aloja font.

You can convert Aloja from TTF/OTF to WOFF2 using a tool like [CloudConvert](https://cloudconvert.com/ttf-to-woff2) or [fonttools](https://github.com/fonttools/fonttools).

If the file is missing, the browser will fall back to the default cursive/sans font.
